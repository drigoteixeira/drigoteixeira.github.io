Atlantic Energias Renováveis.

Website redesign
A2ad, 2015
Project Roles
Information Architecture, UX Analysis, UI Design, Content Strategy

Atlantic Energias Renováveis is a holding company that develops, owns and operates renewable energy projects in Brazil. I was tasked to redesign their existing website, using elements and trends in User Interface. The solution was create a simple navigation and a beautiful homepage/content pages, using video in HTML5 and nice photos that shows the company infrastructure.
